| -------- | :-------------------------------------------------: |

## Intel Xeon E3 1276v3

- Motherboard MSI B85i (Realtek 8111G Gigabit LAN controller)
- CPU Intel E3 1276v3 (with integrated graphics Intel P4600)
- GPU Sapphire RX470D (with RX570 bios)
- RAM G-SKILL DDR3 8G 1866MHz
